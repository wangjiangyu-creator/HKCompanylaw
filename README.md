# Hong Kong Company Law

An academic English static website introducing Hong Kong company law through curated topics, statutory materials, cases, scholarship, and practitioner commentary.

## Local Development

Use the bundled Codex Node/pnpm runtime if Node is not on the default shell path.

```bash
pnpm install
pnpm validate:content
pnpm build
pnpm dev
```

## Scope

This first version is a static research portal. It is not a substitute for legal advice and does not reproduce copyrighted source texts.
